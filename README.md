# whatsapp_ui_clone

A sample WhatsApp UI clone in Flutter.

## Dependancies:

> faker: ^1.0.0

## Setup

To get a copy of this software and start using -

```bash
git clone git@github.com:pritam-patil/whatsapp_ui_clone.git
cd whatsapp_ui_clone
flutter packages get
flutter run
```
